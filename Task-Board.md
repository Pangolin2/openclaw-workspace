## 📝 使用说明

### 创建新任务

```markdown
### Task-XXX: 任务名称
| 属性 | 内容 |
|------|------|
| **状态** | 🟡 TODO |
| **负责人** | Agent名称 |
| **协作Agent** | Agent1, Agent2 |
| **创建时间** | YYYY-MM-DD |
| **优先级** | 🔴 P0 / 🟡 P1 / 🟢 P2 |

**描述**: 任务详细描述

**交付物**:
- [ ] 交付物1
- [ ] 交付物2
```

### 状态流转

```
TODO → In Progress → Review → Done
  ↓
Blocked (阻塞时)
```

### 优先级定义

- 🔴 **P0**: 紧急重要，必须立即处理
- 🟡 **P1**: 重要不紧急，本周完成
- 🟢 **P2**: 一般任务，有空处理
