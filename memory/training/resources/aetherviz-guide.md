## 核心功能

### 1. 一键生成
```bash
# 启动本地服务器
python -m http.server 8080

# 访问 http://localhost:8080
# 输入主题即可生成
```

### 2. 智能识别
- **学科识别**: 自动检测物理/化学/生物/数学/天文/编程
- **渲染方案**: 自动选择 3D / 2D / 混合模式
- **语言适配**: 自动检测中英文

### 3. 渲染能力
- **Three.js**: 专业级3D场景，60fps流畅体验
- **SVG**: 2D函数图像、坐标系、流程图
- **混合模式**: 3D场景 + SVG图表叠加

### 4. 交互功能
- 播放/暂停/单步控制
- 速度滑块调节
- 参数实时调整
- 触摸设备支持（旋转、缩放）

## 使用教程

### 快速开始

1. **克隆仓库**
```bash
git clone https://github.com/andyhuo520/aetherviz-master.git
cd aetherviz-master
```

2. **启动服务器**
```bash
python -m http.server 8080
# 或 npx serve .
```

3. **访问使用**
```
http://localhost:8080
输入主题: 牛顿第二定律
```

### Claude Code 集成
```bash
# 在Claude Code中使用
/aetherviz-master
# 或输入: 三角函数
```
