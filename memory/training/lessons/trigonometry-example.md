## 主题分析

**学科**: 数学  
**难度**: 高中  
**抽象程度**: 中（需要函数图像可视化）

## 可视化方案

**渲染模式**: SVG 2D图表（纯函数图像）  
**配色主题**: 金黄渐变  
**交互元素**: 
- 振幅滑块 (A)
- 频率滑块 (ω)
- 相位滑块 (φ)
- 实时波形绘制

## 教学流程

### 1. 学习目标
- [ ] 理解 y = A·sin(ωx + φ) 的参数意义
- [ ] 观察振幅对波形的影响
- [ ] 观察频率对波形的影响
- [ ] 理解相位移动的概念

### 2. 原理解释
三角函数是描述周期运动的基础工具。通过调整三个参数：
- **振幅A**: 波峰到波谷的高度
- **角频率ω**: 波的密集程度
- **相位φ**: 波的起始位置

### 3. 数学公式

$$y = A \sin(\omega x + \varphi)$$

其中：
- A > 0 为振幅
- ω > 0 为角频率
- φ ∈ [0, 2π) 为初相位

### 4. 交互演示
- 拖动滑块观察波形变化
- 点击"单步"按钮逐步变化
- 调整播放速度观察动态效果

1. 当A增大时，波形会______
2. 当ω增大时，波形会______
3. 当φ增大时，波形会______
