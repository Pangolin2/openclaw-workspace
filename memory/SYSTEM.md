# 🎯 多领域任务调度系统

## 主控Agent (Claw Mate) 职责

作为中央协调者，我根据任务类型将工作路由到相应领域：

| 领域 | 标签 | 处理内容 | 目录 |
|------|------|----------|------|
| 🏠 家庭 | `#home` | 日程、购物、家务、家庭关系 | `memory/home/` |
| 💼 工作 | `#work` | 项目、任务、会议、职业发展 | `memory/work/` |
| 🧠 反思 | `#reflection` | 日记、目标、价值观、成长 | `memory/reflection/` |

## 任务路由规则

### 自动识别关键词

**家庭 (#home)**
- 购物、超市、买菜
- 家人、父母、孩子
- 家务、清洁、维修
- 家庭聚会、节日

**工作 (#work)**
- 项目、任务、deadline
- 会议、邮件、报告
- 技能、学习、职业
- 同事、客户、老板

**反思 (#reflection)**
- 日记、日志、思考
- 目标、计划、梦想
- 感受、情绪、价值观
- 回顾、总结、成长

## 使用方式

### 1. 明确指定领域
> "记录到 #home：明天买牛奶"

### 2. 自动识别
> "帮我安排周末家庭聚会" → 自动路由到 #home

### 3. 跨领域整合
> "最近工作压力大，影响家庭关系" → 同时参考 #work 和 #home
