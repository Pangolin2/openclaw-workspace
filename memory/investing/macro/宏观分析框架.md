## 3️⃣ 北向资金流向

### 跟踪维度
| 维度 | 数据 | 分析 |
|------|------|------|
| 日净流入 | _亿 | 单日大幅流入/流出需关注 |
| 周净流入 | _亿 | 短期趋势判断 |
| 月净流入 | _亿 | 中期趋势判断 |
| 行业偏好 | _ | 外资看好哪些板块 |
| 个股偏好 | _ | 外资重仓股变化 |

### 关键观察
- **持续流入**: 外资看好，可跟随
- **持续流出**: 外资撤离，需谨慎
- **单日异动**: 可能受事件驱动，需分析原因

### 分析框架
- **经济数据向好** → 企业盈利改善 → 利好股市
- **经济数据走弱** → 政策刺激预期 → 关注政策受益板块
- **通胀上升** → 紧缩预期 → 利空股市
- **通缩风险** → 宽松预期 → 关注政策动向
