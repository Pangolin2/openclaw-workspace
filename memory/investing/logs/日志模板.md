## 🎯 复盘要点

### 每笔交易后必须回答
1. **为什么买/卖？** - 逻辑是否清晰？
2. **计划是什么？** - 目标价和止损点是否合理？
3. **执行如何？** - 是否按计划执行？有无冲动？
4. **结果怎样？** - 盈利/亏损的原因是什么？
5. **学到什么？** - 有什么经验教训？

### 定期复盘检查清单
- [ ] 是否遵守了投资原则？
- [ ] 是否有冲动交易？
- [ ] 止损纪律执行如何？
- [ ] 仓位管理是否合理？
- [ ] 情绪控制是否到位？
- [ ] 分析框架是否需要优化？
