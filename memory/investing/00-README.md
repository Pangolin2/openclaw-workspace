## 🎯 使用流程

### 阶段1: 宏观扫描 (每周)
1. 检查美联储/央行政策动向
2. 分析北向资金流向
3. 评估市场整体情绪
4. 确定仓位策略

### 阶段2: 行业筛选 (每月)
1. 筛选政策支持/景气度向上的行业
2. 分析产业链上下游
3. 评估竞争格局
4. 确定重点关注行业

### 阶段3: 个股研究 (持续)
1. 从重点行业中筛选标的
2. 使用个股分析清单深入研究
3. 计算估值，确定安全边际
4. 建立观察清单，等待买点

### 阶段4: 交易执行 (按计划)
1. 检查买点是否触发
2. 计算仓位，执行买入
3. 设置止损/止盈点
4. 记录交易日志

### 阶段5: 复盘总结 (定期)
1. 每日更新投资日志
2. 每周回顾持仓逻辑
3. 每月分析决策质量
4. 持续优化分析框架

## 📝 核心原则

### 1. 不预测，只应对
- 不猜测涨跌，只设定触发条件
- 买入前确定止损点
- 计划你的交易，交易你的计划

### 2. 能力圈原则
- 只投资理解的行业和公司
- 不懂的不碰（如复杂的金融衍生品）
- 持续扩展能力圈

### 3. 安全边际
- 估值留有安全边际
- 仓位控制分散风险
- 单只股票不超过总仓位20%

### 4. 长期视角
- 关注企业内在价值，而非短期波动
- 避免频繁交易
- 让复利发挥作用
