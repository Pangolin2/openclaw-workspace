# 行业分析模板

> 中观分析：识别景气度向上的行业

---

## 📋 分析清单

### 一、行业基本面

#### 1. 行业空间
- [ ] 行业规模：当前___亿，年复合增长率___%
- [ ] 渗透率：当前___%，理论上限___%
- [ ] 生命周期：导入期/成长期/成熟期/衰退期
- [ ] 国际对比：与发达国家差距，发展空间

#### 2. 竞争格局
- [ ] 行业集中度：CR5 = ___%
- [ ] 竞争态势：完全竞争/垄断竞争/寡头垄断
- [ ] 进入壁垒：高/中/低（技术/资金/牌照）
- [ ] 替代品威胁：有无颠覆性技术风险

#### 3. 盈利能力
- [ ] 平均毛利率：___%
- [ ] 平均净利率：___%
- [ ] ROE水平：___%
- [ ] 盈利趋势：上升/持平/下降

### 二、政策环境

#### 1. 政策支持力度
- [ ] 国家政策：是否有专项支持政策
- [ ] 地方政府：是否有补贴/优惠
- [ ] 产业规划：是否列入重点发展领域
- [ ] 政策持续性：短期刺激 or 长期战略

#### 2. 监管风险
- [ ] 监管态度：鼓励/规范/限制
- [ ] 政策变化：近期是否有重大政策调整
- [ ] 合规要求：准入门槛/环保要求等

**政策敏感度评估**:
| 敏感度 | 行业特征 | 例子 |
|--------|---------|------|
| 🔴 高 | 强监管行业 | 医药、教育、金融、地产 |
| 🟡 中 | 周期性行业 | 钢铁、煤炭、有色 |
| 🟢 低 | 消费/科技 | 食品饮料、软件 |

### 三、景气度判断

#### 1. 需求端
- [ ] 下游需求：增长/持平/萎缩
- [ ] 价格趋势：上涨/持平/下跌
- [ ] 库存水平：高位/正常/低位
- [ ] 订单情况：饱满/正常/不足

#### 2. 供给端
- [ ] 产能利用率：___%
- [ ] 新产能投放：有无大规模新增产能
- [ ] 行业库存：去库存/补库存

#### 3. 产业链位置
```
上游原材料 → 中游制造 → 下游应用
    ↓           ↓          ↓
  成本        加工费      终端价格
```

- [ ] 议价能力：对上游/对下游
- [ ] 成本传导：能否将成本上涨转嫁给下游

### 四、估值水平

| 指标 | 当前 | 历史平均 | 分位数 | 判断 |
|------|------|---------|--------|------|
| PE | _ | _ | _% | 高/中/低 |
| PB | _ | _ | _% | 高/中/低 |
| PS | _ | _ | _% | 高/中/低 |

### 五、资金流向

- [ ] 北向资金：近期增持/减持
- [ ] 公募基金：配置比例变化
- [ ] 主力资金：板块净流入/流出
