# 01-投资原则与纪律

> 投资最重要的是不亏大钱，其次才是赚小钱。

**⚠️ 免责声明**: 本文件记录个人投资纪律，不构成投资建议。

---

## 🛡️ 核心原则

### 原则1: 不懂不投
- [ ] 我能用一句话说明这家公司的商业模式吗？
- [ ] 我知道这家公司如何赚钱吗？
- [ ] 我了解这个行业的竞争格局吗？
- [ ] 我能大致判断这家公司的合理估值吗？

**如果以上任一答案为"否"，则不投资。**

### 原则2: 安全边际
- [ ] 买入价格是否有足够的安全边际？
- [ ] 最坏情况下，我会亏损多少？
- [ ] 这个亏损我能接受吗？

**永远不要支付过高的价格。**

### 原则3: 分散投资
- [ ] 单只股票仓位不超过20%
- [ ] 单个行业仓位不超过30%
- [ ] 至少持有5只不同行业的股票

**不要把所有鸡蛋放在一个篮子里。**

### 原则4: 长期持有
- [ ] 我打算持有这只股票至少1年吗？
- [ ] 如果明天停牌1年，我会害怕吗？
- [ ] 我关注的是企业价值，而非股价波动？

**频繁交易是收益的敌人。**

---

## 🚫 禁止清单

### 绝不做的行为
- ❌ 追高买入（涨幅>5%当日不买）
- ❌ 满仓操作（保留至少20%现金）
- ❌ 杠杆融资（不用融资融券）
- ❌ 听消息炒股（不跟风"内幕消息"）
- ❌ 亏损补仓（不摊低成本，该止损就止损）
- ❌ 短期预测（不预测明天涨跌）

### 绝不投资的标的
- ❌ ST股票
- ❌ 市值<50亿的小盘股（流动性风险）
- ❌ 连续3年亏损的公司
- ❌ 财务造假嫌疑的公司
- ❌ 自己完全不懂的行业（如复杂的生物科技）

## ✅ 必须执行的纪律

### 买入前检查清单
- [ ] 已完成基本面分析（[[个股分析清单]]）
- [ ] 已计算估值，价格有安全边际
- [ ] 已确定止损点（跌破X元或X%止损）
- [ ] 已确定目标仓位（不超过总资金Y%）
- [ ] 已写入投资日志（买入理由）

### 持有期检查（每月）
- [ ] 公司基本面是否发生变化？
- [ ] 买入逻辑是否仍然成立？
- [ ] 估值是否变得过高？
- [ ] 是否有更好的投资机会？

### 卖出条件（任一触发即卖出）
- [ ] 触及止损点（纪律性止损）
- [ ] 基本面恶化（逻辑被破坏）
- [ ] 估值严重高估（泡沫化）
- [ ] 发现更好的投资机会（机会成本）

---

## 🧠 心理建设

### 面对亏损
- 亏损是投资的一部分，坦然接受
- 止损是为了防止小亏变大亏
- 不要因为面子而不肯认错
- 记录错误，复盘学习

### 面对盈利
- 不要被短期盈利冲昏头脑
- 评估是运气还是能力
- 不要因此提高风险偏好
- 定期提取部分利润

### 面对市场波动
- 短期波动无法预测，不必焦虑
- 关注企业内在价值，而非股价
- 利用市场情绪（别人恐惧我贪婪）
- 保持独立判断，不随波逐流

## 📊 仓位管理规则

### 市场环境判断
| 市场状态 | 仓位建议 | 操作风格 |
|---------|---------|---------|
| 牛市初期 | 70-80% | 积极买入 |
| 牛市中期 | 60-70% | 持有为主 |
| 牛市末期 | 30-40% | 逐步减仓 |
| 熊市初期 | 20-30% | 观望为主 |
| 熊市中期 | 20%以下 | 极少操作 |
| 熊市末期 | 逐步加仓 | 精选标的 |

### 个股仓位分配
- 核心持仓（深入研究）: 10-20%
- 卫星持仓（一般研究）: 5-10%
- 观察试仓（初步关注）: 2-5%

---

## 🔄 定期复盘

### 每周
- 更新宏观数据
- 检查持仓逻辑
- 记录市场观察

### 每月
- 分析持仓表现
- 更新观察清单
- 评估投资策略

### 每季度
- 深度复盘决策质量
- 更新能力圈范围
- 优化分析框架

### 每年
- 总结年度收益/亏损
- 分析主要决策得失
- 制定下年投资计划
